## 關於

「食在有愛」（諧音「實在有愛」）是國立台南大學在職碩班團隊與 NPOChannel 公益整合平台合作的公益消費計畫。我們相信，真正的關懷不只是理念，而是可以被實踐的行動。透過販售在地肉乾／果乾禮盒，建立「售一即捐」的直接捐贈循環。
